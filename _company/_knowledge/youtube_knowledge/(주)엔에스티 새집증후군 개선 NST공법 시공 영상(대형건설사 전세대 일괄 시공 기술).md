# [[(주)엔에스티 새집증후군 개선 NST공법 시공 영상(대형건설사 전세대 일괄 시공 기술)]]

## 📌 Brief Summary
새 집인데 왜 눈이 시리고, 머리가 아플까요?

## 📖 Core Content
보이지 않는 실내 유해물질, 이제는 단순한 방향제가 아닌 과학적인 해결이 필요합니다.
㈜엔에스티는 ‘새집증후군’의 원인을 근본적으로 해결하기 위해
직접 개발한 3단계 솔루션을 제공합니다.
🔹 NST-S100  / 일반공간 오염지역 지속 제거
눈에 보이지 않는 유해가스를 흡착하고 광분해하는 가시광 반응형 투명 광촉매 코팅제
햇빛/실내 조명만으로도 자가세정(Self-Cleaning) 작용이 반복
🔹 NST-S200 / 대량방출 오염지역 지속 제거
유해물질이 방출되는 자재 표면을 침투·차단 피막 형성으로 근본적인 발생원 봉쇄 효과
🔹 NST-F100 / 시공취약 오염지역 제거 보조
최종 단계에서 마무리 솔루션 잔류 오염물질을 흡착
