사장님, 지시 감사합니다. 현재 저희 에이전트 시스템은 목표 달성을 위한 **구조적 준비(워크플로우 및 도구)**는 완료된 상태입니다. 하지만 가장 핵심적인 **데이터 연결고리**가 끊겨서 실질적인 분석을 진행할 수가 없습니다. 📊

현재 API 키와 채널 ID 정보가 없기 때문에, 아무리 완벽한 기획서를 짜도 실제 트렌드나 경쟁사의 데이터를 끌어올 수 없습니다. 데이터 없이 추측하는 것은 '감'에 의존하는 것과 같고, 저희의 목표는 **숫자 기반의 확신**을 얻는 것입니다.

따라서 지금 당장 할 수 있는 가장 중요한 액션은 기술적인 준비가 아니라, 시스템 설정을 완료하는 것입니다.

---
🚨 **[필수 안내] 데이터 연결 요청** 🚨

지금 분석을 시작하려면 YouTube API 키와 채널 ID가 필요해요. 헤더 우측 "👥 직원 에이전트 보기" 버튼 → YouTube 카드 ⚙️ 클릭 → API 키와 채널 ID를 입력 후 다시 요청해주세요.
---

데이터만 확보되면, 즉시 다음 순서로 업무를 진행하겠습니다:

1.  **`youtube_account` 실행:** 모든 도구에 필요한 자격 증명을 한 번에 시스템 메모리에 주입합니다. (최초 1회)
2.  **`comment_harvester` 실행:** 감시 채널에서 시청자가 실제로 쓰는 '후크 키워드'를 수집하여 `memory.md`에 누적합니다.
3.  **`competitor_brief` 실행:** 경쟁사 인기 영상의 성공 패턴을 추출하고, 이를 기반으로 후크가 강한 영상 기획안 3개를 작성하겠습니다.

📊 평가: 대기 — 핵심 자격 증명(API Key/Channel ID)이 부족하여 모든 분석 도구 사용 및 결과 도출이 불가능합니다.
