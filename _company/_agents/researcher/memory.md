### Connect AI 시스템 핵심 개념 (채널에서 가르치는 내용)

**Brain-GitHub 동기화 구조:**
- 로컬 지식 폴더 ↔ GitHub 자동 동기화
- 에이전트 메모리 자동 축적 (append-only)
- Brain Pack 주입 시스템 (POST /api/brain-inject)
- P-Reinforce 구조화 학습

**1인 기업 OS 구조:**
- CEO 에이전트가 오케스트레이션
- 7+ 전문 에이전트 (YouTube, Instagram, Designer, Developer, Business, Secretary, Writer, Researcher)
- AUTONOMY_LEVEL 개념 (0-3단계)
- skills/ 폴더: 검증된 패턴 자동 주입

**기술 스택:**
- Ollama (로컬 LLM 실행)
- gemma4:e4b (RTX 3060 12GB 최적 모델)
- VS Code 익스텐션 or 데스크톱 앱 형태
- GitHub 연동 (두 번째 뇌)
